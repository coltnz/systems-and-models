## Problem
The alpha needs a single, reproducible end-to-end demo proving the whole loop:
supply a transcript → generate a draft pack (mock adapter offline) → validate → review/edit
→ save a reviewed pack → ask a grounded tutor question that cites reviewed anchors or refuses.

## Where to look
- All prior beads' public surfaces; `spec/examples/spaced-repetition-talk.learning-pack.json`
  as the shape target for the generated/reviewed pack.

## Proposed change (sketch)
`packages/e2e` (+ root README section):
- One scripted demo **source** (a short CC/authored transcript `.md`) committed under the
  package's fixtures.
- A script (or test) that runs ingest → extract(mock) → validate → a scripted review →
  save reviewed pack, writing artifacts to a temp/`.systems-and-models/` dir.
- Assertions: draft pack validates; reviewed pack satisfies the publish invariant; a sample
  in-scope tutor question returns a cited answer; an out-of-scope one refuses.
- Commit the **one generated pack** and the **one reviewed pack** as example artifacts
  (or generate them deterministically in CI).
- **README**: prerequisites (Node 22), install, `npm test`, how to run the local server +
  web UI, where to put `OPENAI_*` env vars (and that the demo runs fully offline on the mock).

## Definition of done
- `npm test` runs the e2e green offline (mock adapter), producing the demo artifacts.
- README lets a fresh clone reach the working alpha (UI + tutor) following the steps.
- The scripted demo matches the alpha goal sentence end-to-end.

## Gates
- Full root gate green: `npm install && npm run build && npm run lint && npm test && npm run typecheck`.

## Constraints
- Demo must run with no network / no API key (mock). OpenAI path documented but optional.
- Source must be CC-licensed or authored (no copyright risk in the committed fixture).
