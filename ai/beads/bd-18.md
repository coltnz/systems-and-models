# bd-18 — One-click demo seed

- **Status:** done
- **Type:** implementation
- **Depends on:** bd-8, bd-10, bd-17
- **Blocks:** —

## Problem

The alpha can be proven by `npm run demo` and by manually pasting
`packages/e2e/fixtures/circuit-breaker.md`, but the web UI still opens as an
empty form. A first-time reviewer has to know which fixture to copy before they
can see the source-to-draft loop.

## Scope

Add a one-click demo path to `packages/web` that uses the same original
CC-BY-4.0 circuit-breaker source and the existing `createSourceAndDraft` API
flow. This is a UI convenience only: no server, schema, validator, or extraction
contract changes.

## Definition of done

- New-source form includes a demo button.
- Demo button creates a draft via the same API path as a pasted source.
- Web tests cover the demo path and assert the expected source metadata/content.
- Existing web build/test/typecheck stays green.

## Notes

Implemented in `packages/web/src/demo-source.ts` and
`packages/web/src/components/NewSourceForm.tsx`. The demo source is duplicated in
the web package instead of imported from `@sam/e2e`, keeping package boundaries
simple and avoiding raw-file loader configuration.
